
# Student Exam Portal Mobile App

A **React Native mobile application** designed for students to register for exams, select subjects, and manage exam records efficiently. Built as part of a coursework project to showcase **React Native skills, UI/UX design, and mobile app development best practices**.

---

## Features

* **User Authentication:** Login system with registration number and password validation.
* **Subject Selection:** Students can select multiple subjects for registration.
* **Exam Records:** View selected subjects, exam types, and payment details.
* **Interactive UI:** Smooth navigation between screens using **React Navigation**.
* **State Management:** Efficient handling with `useState` and `useEffect`.
* **Clean & Modular Code:** Reusable components, proper folder structure, and styling using `StyleSheet`.
* **User-Friendly Design:** Dark theme with highlighted selections for better UX.

---

## 🛠 Tech Stack

* **React Native** – Mobile app framework
* **React Navigation** – Screen navigation
* **useState & useEffect** – State management
* **StyleSheet** – Styling
* **AsyncStorage**  – Local data storage
